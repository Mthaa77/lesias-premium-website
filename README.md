# Lesia's Educational Services — Premium Website

Premium static website for Lesia's Educational Services, built as a high-converting admissions landing page for accredited ECD training.

## Features

- Cinematic premium hero section
- Interactive ECD pathway quiz
- Level 4 and Level 5 programme showcases
- Programme comparison section
- Student support and career pathway sections
- Admissions steps
- FAQ accordion
- WhatsApp, phone, and email CTAs
- Fully static HTML/CSS/JS in a single `index.html` file

## Run locally

Open `index.html` in your browser, or serve it with:

```bash
python3 -m http.server 3000
```

Then visit `http://localhost:3000`.

## Deploy

This can be deployed to GitHub Pages, Vercel, Netlify, Firebase Hosting, or any normal static hosting provider.
